Controllers

É na pasta Controllers que ficarão todos os seus controllers, ou seja, as classes responsáveis por coordenar as repostas às requisicoes.

Por exemplo, comumente o seu controller vai acessar o seu banco de dados através de uma model e então retornar esses dados como um json ou passa-los para uma view que irá preencher o html a ser retornado para o usuário.

Neste arquivo você irá configurar seus filtros de rota. Filtros de rota são classes com um único método "aplicar" no qual você define a lógica de cada filtro.
