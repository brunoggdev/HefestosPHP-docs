Forja

A forja é a ferramenta de interface de linha de comando (CLI) do HefestosPHP para interagir com o framework. Ou seja, é uma ferramenta que deve ser acessada pelo terminal para realizar comandos que te auxiliam no desenvolvimento do seu aplicativo. Esses comandos são:

  • Iniciar
  • Criar
  • Migrar
  • Testar
  • Fornalha

Comando Iniciar

Esse comando inicia o servidor de desenvolvimento do Hefestos, podendo ser acessado na url 'localhost:8080', onde 8080 é o valor padrão da porta, esta que pode ser configurada informando um valor opcional. Por exemplo:

php forja iniciar

php forja iniciar 8081

Novamente, ambas versões funcionam de modo identico, apenas com a diferença de que a porta padrão é 8080 caso outra não seja informada.

Comando Criar

Esse comando permite criar tipos pré definidos de arquivos em seus devidos repositórios. São eles: controllers, models, filtros, tabelas e composer. Pode ser usado da seguinte maneira:

php forja criar model Pets

Comando Migrar

O comando migrar executa as tabelas definidas no seu diretório de tabelas. O comando irá executar todas as tabelas no diretório ou apenas a especificada. Há ainda um indicador "zero" para executar a tabela "do zero", ou seja, mesmo que ela já exista, será deletada e recriada. Por exemplo, todas as opções abaixo são válidas:

php forja migrar // executa todas as tabelas

php forja migrar zero // executa todas as tabelas do zero

php forja migrar pets // executa apenas a tabela pets

php forja migrar pets zero // executa apenas a tabela pets do zero

Comando Testar

Similarmente ao comando migrar, o comando testar executa os testes definidos no seu diretório de testes. O comando irá também executar todos os testes no diretório ou apenas os especificados. Ou seja, você pode informar um nome de arquivo ou mesmo um subdiretório inteiro para que apenas os seus testes sejam executado. Desse modo:

php forja testar

php forja testar subpasta

php forja testar subpasta/testes_especificos.php

Comando Fornalha

O comando fornalha permite ao desenvolvedor ter acesso à toda sua aplicação por meio da linha de comando e executar código PHP nesse contexto. Com isso, é possível a realização de diversas tarefas como acessar arquivos e constantes, comunicação com o banco de dados ou testar funcionalidades, como exemplificado abaixo:

php forja fornalha
