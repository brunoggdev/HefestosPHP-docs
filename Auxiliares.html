Arquivo Auxiliares

O HefestosPHP possui diversas funções auxiliares (que você pode conferir aqui), mas também te encoraja a criar suas próprias! Você pode criar as funções que desejar neste arquivo, bem como definir constantes, e tudo estará disponível em todo seu app.

Por exemplo, você deve estar utilizando a estrutura de componentes header e footer, certo? Podemos criar uma função auxiliar para concatenar esses componentes a view desejada e utiliza-la no controller invés da função view() diretamente.

Mas por que parar por aí? Talvez você queira que cada view tenha acesso à um arquivo .js específico pra ela, e quem sabe até suas modais específicas? Podemos agrupar tudo isso numa única função

  • Passando dados para view
  • Sintaxe alternativa do PHP
  • Montando uma view

Passando dados para view

Uma view sempre deve ser acessada pela função view(), que aceita como primeiro parametro o nome da view desejada (sem a extensão .php) e, opcionalmente, um array associativo de dados indicando as variaveis disponíveis na view.

Note que, no fim, a função view apenas retorna uma string do seu conteúdo HTML. Isso quer dizer que todas as operações com strings podem ser utilizadas com seu retorno, como concatenação. Confira Auxiliares para um exemplo interessante.
