Pasta Models

Na pasta Models ficarão, claro, os seus Models, ou seja, a camada do seu projeto responsável por acessar o seu banco de dados.

Cada Model está diretamente associada a uma de suas tabelas no banco de dados e irá interagir com ela para realizar as operações desejadas. Para tal, todas as Models já possuem métodos herdados para auxiliar nessa interação.

Você pode também, claro, criar seus próprios métodos dentro de cada Model e utilizar neles esses métodos auxiliares para facilitar ainda mais as operações. Confira os exemplos abaixo.

  • Criando um Model
  • Inserindo dados
  • Busncando dados
  • Atualizando dados
  • Deletando dados

Busncando dados

Ao buscar dados do banco, as models irão retornar cada linha representada como um array (padrão) ou objeto.

Para buscar apenas uma linha especifica utilize o método buscar() informando o id da linha desejada. Opcionalmente, você pode informar uma coluna especifica para retornar diretamente seu valor invés de um array.
